# 人工智能导论实验报告：DDPM扩散模型研究与复现
本文件是华中科技大学计算机科学与技术专业《人工智能导论》课程实验报告，围绕**去噪扩散概率模型（DDPM）** 展开系统性的理论研究与工程复现，完整梳理了生成式AI的技术演进脉络，完成了扩散模型的原理拆解、代码复现、效果验证与行业分析。

## 文档核心内容概述
本报告以DDPM扩散模型为核心，从技术发展史、行业现状、原理复现、优化方向四大维度展开，完整覆盖了生成式模型从理论到实践的全链路内容，核心板块如下：
1.  **生成式模型发展历程梳理**
    系统追溯了生成式模型的完整演进路径：从1990-2000年的隐马尔可夫模型（HMM）、贝叶斯网络、玻尔兹曼机等经典概率模型，到深度学习时代受限玻尔兹曼机（RBM）、深度信念网络（DBN）、变分自编码器（VAE）、生成对抗网络（GAN）的技术突破，再到Transformer架构推动的大模型与跨模态生成发展，最终聚焦到DDPM为代表的扩散模型的兴起，厘清了各阶段技术的核心特点与演进逻辑。

2.  **生成式AI国内外发展现状分析**
    分别拆解了全球与国内生成式AI的发展格局：国外部分聚焦OpenAI的GPT系列、DALL-E等核心技术突破，企业级落地应用案例，全球市场规模增长趋势，以及日韩等国的AI政策布局与伦理监管现状；国内部分分析了我国生成式AI的市场规模、用户体量、行业融合趋势，以及旷视科技、百度、阿里云等头部企业的技术研发与商业化落地成果，同时点明了AI技术发展面临的伦理与安全挑战。

3.  **DDPM模型原理详解与工程复现**
    这是报告的核心板块，先对比了DDPM与自编码器（AE）、VAE、GAN等主流生成模型的本质差异，拆解了DDPM“前向逐步加噪、反向逐步去噪”的核心原理与数学逻辑；随后基于AutoDL云平台完成了DDPM的完整代码复现，分别在**MNIST手写数字数据集**与**CelebA人脸数据集**完成了模型训练与图像生成实验，通过多轮迭代验证了模型的学习能力与生成效果，完整呈现了扩散模型从理论到落地的实现流程。

4.  **扩散模型优化方向总结**
    针对原生DDPM采样速度慢、算力开销大等痛点，梳理了扩散模型的主流优化路径，包括采样加速技术、扩散过程设计优化、似然估计方法改进、轻量级微调方案（P-Tuning V2、LoRA）、任意分布桥接技术等多个维度，为扩散模型的后续改进与落地应用提供了技术参考。

5.  **研究总结与行业展望**
    复盘了本次实验的理论学习与工程实践成果，结合Sora等前沿视频生成技术，分析了当前生成式AI行业的竞争格局与发展前景；同时强调了生成式AI技术快速发展过程中，必须重视的伦理规范、数据安全与监管合规问题，明确了技术创新与安全发展并行的核心观点。

## 文档核心价值
本报告完成了生成式AI从理论溯源、原理拆解、工程复现到行业分析的完整闭环，既以通俗易懂的方式厘清了DDPM扩散模型的核心逻辑，也提供了可复现的实验流程与结果验证，同时结合行业前沿动态与国内外发展现状，对生成式模型的发展做了全面的分析，是扩散模型与生成式AI入门学习、课程研究的完整参考资料。

本报告仅用于课程实验与学术学习研究，不涉及任何商业用途。
